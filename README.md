## A simple TODO list app just to demonstrate React router, fastify and postgres usage

#### Extremely simple design. It has 3 endpoints

GET: / (To fetch all items)

POST: / (To insert a new item)

DELETE: /:id (Deletes a specific item)

React side uses a simple BrowserRouter implementation with all actions abstracted into the useTodos hook.
